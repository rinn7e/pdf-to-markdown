# មាតិកា

(ច្បាប់ស្តីពីការរៀបចំអង្គការតុលាការ មានចំនួន ៨ជំពូក និង ៩១មាត្រា)

- **ជំពូកទី១ បទប្បញ្ញត្តិទូទៅ** (មាត្រា១ ដល់មាត្រា១១)
- **ជំពូកទី២ សាលាដំបូង និងអយ្យការអមសាលាដំបូង** (មាត្រា១២ ដល់មាត្រា៣៤)
    - **ផ្នែកទី១** សាលាដំបូង
        - **អនុផ្នែកទី១** ការរៀបចំ និងការប្រព្រឹត្តទៅរបស់សាលាដំបូង (មាត្រា១២ ដល់មាត្រា១៨)
        - **អនុផ្នែកទី២** សមត្ថកិច្ច (មាត្រា១៩ ដល់មាត្រា២៨)
    - **ផ្នែកទី២** អយ្យការអមសាលាដំបូង (មាត្រា២៩ ដល់មាត្រា៣១)
    - **ផ្នែកទី៣** លេខាធិការដ្ឋានរដ្ឋបាលសាលាដំបូង (មាត្រា៣២ ដល់មាត្រា៣៤)
- **ជំពូកទី៣ សាលាឧទ្ធរណ៍ និងមហាអយ្យការអមសាលាឧទ្ធរណ៍** (មាត្រា៣៥ ដល់មាត្រា៥៤)
    - **ផ្នែកទី១** សាលាឧទ្ធរណ៍
        - **អនុផ្នែកទី១** ការរៀបចំ និងការប្រព្រឹត្តទៅរបស់សាលាឧទ្ធរណ៍ (មាត្រា៣៥ ដល់មាត្រា៤០)
        - **អនុផ្នែកទី២** សមត្ថកិច្ច (មាត្រា៤១ ដល់មាត្រា៤៨)
    - **ផ្នែកទី២** មហាអយ្យការអមសាលាឧទ្ធរណ៍ (មាត្រា៤៩ ដល់មាត្រា៥១)
    - **ផ្នែកទី៣** អគ្គលេខាធិការដ្ឋានរដ្ឋបាលសាលាឧទ្ធរណ៍ (មាត្រា៥២ ដល់មាត្រា៥៤)
- **ជំពូកទី៤ តុលាការកំពូល និងមហាអយ្យការអមតុលាការកំពូល** (មាត្រា៥៥ ដល់មាត្រា៧៣)
    - **ផ្នែកទី១** តុលាការកំពូល
        - **អនុផ្នែកទី១** ការរៀបចំ និងការប្រព្រឹត្តទៅរបស់តុលាការកំពូល (មាត្រា៥៥ ដល់មាត្រា៦០)
        - **អនុផ្នែកទី២** សមត្ថកិច្ច (មាត្រា៦១ ដល់មាត្រា៦៧)
