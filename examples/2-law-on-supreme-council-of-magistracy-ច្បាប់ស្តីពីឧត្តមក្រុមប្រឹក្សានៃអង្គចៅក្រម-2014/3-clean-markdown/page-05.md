# បទប្បញ្ញត្តិទូទៅ

**មាត្រា ១.-**
ច្បាប់នេះមានគោលដៅបង្កើត និងកំណត់អំពីការរៀបចំ និងការប្រព្រឹត្តទៅនៃឧត្តមក្រុមប្រឹក្សានៃអង្គចៅក្រម ក្នុងគោលបំណងជួយព្រះមហាក្សត្រ ដើម្បីធានាឯករាជ្យនៃអំណាចតុលាការស្របតាមរដ្ឋធម្មនុញ្ញនៃព្រះរាជាណាចក្រកម្ពុជា ។

**មាត្រា ២.-**
ច្បាប់នេះ មានវិសាលភាពអនុវត្តទៅលើចៅក្រម និងព្រះរាជអាជ្ញា ទូទាំងព្រះរាជាណាចក្រកម្ពុជា លើកលែងតែមានច្បាប់ដោយឡែកកំណត់ផ្សេងពីនេះ ។

**មាត្រា ៣.-**
វាក្យស័ព្ទបច្ចេកទេសដែលប្រើក្នុងច្បាប់នេះ មានន័យដូចខាងក្រោម៖
- **ចៅក្រម** សំដៅដល់ចៅក្រមអាសនៈ រួមមានជាអាទិ៍ ចៅក្រមជំនុំជម្រះក្តី និងចៅក្រមស៊ើបសួរ ។
- **ព្រះរាជអាជ្ញា** សំដៅដល់ព្រះរាជអាជ្ញារង ព្រះរាជអាជ្ញា អគ្គព្រះរាជអាជ្ញារង អគ្គព្រះរាជអាជ្ញា ។
- **អង្គចៅក្រម** សំដៅដល់ចៅក្រម និងព្រះរាជអាជ្ញា ។

# ជំពូកទី ២
# ការរៀបចំឧត្តមក្រុមប្រឹក្សានៃអង្គចៅក្រម

## ផ្នែកទី ១
## សមាសភាពឧត្តមក្រុមប្រឹក្សានៃអង្គចៅក្រម

**មាត្រា ៤.-**
ឧត្តមក្រុមប្រឹក្សានៃអង្គចៅក្រមស្ថិតក្រោមព្រះរាជាធិបតីភាពនៃព្រះមហាក្សត្រ និងរួមមានសមាសភាពដូចខាងក្រោម ៖

១- រដ្ឋមន្ត្រីក្រសួងយុត្តិធម៌ ជាសមាជិក

២- ប្រធានតុលាការកំពូល ជាសមាជិក
